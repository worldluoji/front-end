DOM Selector Tool User Guide

Introduction

DOM Selector Tool is a browser extension that allows you to select DOM elements on web pages, save them, and export their text content. It helps developers and users quickly extract information from web pages.

Features

  • Select DOM Elements: Click or use the context menu to select elements on a web page.
  • Save Selected Content: Save selected DOM elements and their text content to local storage.
  • Export Data: Export saved content as JSON.

How to Use

  1. Install the Extension:

    Download and install DOM Selector Tool from the browser extension store.

  2. Launch the Extension:

    Click the extension icon in the browser toolbar to open the popup window.

  3. Select DOM Elements:

    Right-click on the target element on the web page and choose "Select with DOM Selector Tool".

  4. View and Manage Data:

    Click the extension icon to access the management page and view saved DOM elements and their text content.

  5. Export Data:

    On the management page, click the "Export" button to save the data as a JSON.

Frequently Asked Questions

  • Why can't I select certain elements?

    It may be because the element is hidden or restricted by browser security policies.

  • How do I clear saved data?

    On the management page, click the "Clear Data" button.

DOM Selector Tool 使用说明

简介

DOM Selector Tool 是一款浏览器插件，用于选择网页中的 DOM 元素，保存它们，并导出其文本内容。它可以帮助开发者和普通用户快速提取网页中的信息。

功能概述

  • 选择 DOM 元素：通过点击或右键菜单选择网页中的元素。
  • 保存选中内容：将选中的 DOM 元素及其文本内容保存到本地缓存。
  • 导出数据：支持将保存的内容导出为 JSON。

如何使用

  1. 安装插件：

    从浏览器扩展商店下载并安装 DOM Selector Tool。

  2. 启动插件：

    点击浏览器工具栏中的插件图标，打开弹出窗口。

  3. 选择 DOM 元素：

    在网页上右键单击目标元素，选择“使用 DOM Selector Tool 选择”选项。

  4. 查看和管理数据：

    点击插件图标，进入管理页面，查看已保存的 DOM 元素及其文本内容。

  5. 导出数据：

    在管理页面中，点击“导出”按钮，将数据保存为 JSON。

常见问题

  • 为什么无法选择某些元素？

    可能是因为该元素被隐藏或受浏览器安全策略限制。

  • 如何清除保存的数据？

    在管理页面中，点击“清除数据”按钮即可。
