CSS 断行属性实例

word-break 与 overflow-wrap 的对比与使用

本页面展示了CSS中控制文本断行的两个重要属性：word-break 和 overflow-wrap。通过对比不同属性值的效果，您可以直观地了解它们如何影响文本的换行行为。

1. 默认情况 (normal)

word-break: normal;
overflow-wrap: normal;

normal 是默认值。英文只在空格处换行，长单词会溢出容器。中文可以在任意字符间换行。

2. word-break: break-all

word-break: break-all;
ThisisaverylongEnglishwordthatdoesn'thavespaces 这是一个普通的中文句子没有任何空格。Anotherlongword: Supercalifragilisticexpialidocious
break-all 允许在任意字符间断行，防止文本溢出。适合处理长URL或代码字符串。

3. word-break: keep-all

word-break: keep-all;
ThisisaverylongEnglishwordthatdoesn'thavespaces 这是一个普通的中文句子没有任何空格。Anotherlongword: Supercalifragilisticexpialidocious
keep-all 对CJK文本不允许在字符间换行，只能在空格或标点处换行。英文行为同normal。

4. overflow-wrap: break-word

overflow-wrap: break-word;
ThisisaverylongEnglishwordthatdoesn'thavespaces 这是一个普通的中文句子没有任何空格。Anotherlongword: Supercalifragilisticexpialidocious
break-word 先尝试正常换行，如果不行则在单词内换行。适合自然语言文本。

5. word-break: break-all + overflow-wrap

word-break: break-all;
overflow-wrap: break-word;
ThisisaverylongEnglishwordthatdoesn'thavespaces 这是一个普通的中文句子没有任何空格。Anotherlongword: Supercalifragilisticexpialidocious
同时使用 break-all 和 break-word，效果以word-break为主，因为break-all更激进。

6. 组合使用场景

word-break: break-all;
overflow-wrap: anywhere;
ThisisaverylongEnglishwordthatdoesn'thavespaces 这是一个普通的中文句子没有任何空格。Anotherlongword: Supercalifragilisticexpialidocious
anywhere 是overflow-wrap的新值，行为类似break-word，但允许在任意点断行，即使是为了最小化内容溢出。
