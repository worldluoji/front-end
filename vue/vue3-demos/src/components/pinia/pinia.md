# pinia
Pinia是一个全新的Vue状态管理库，是Vuex的代替者:

- Vue2 和 Vue3 都能支持
- 抛弃传统的 Mutation ，只有 state, getter 和 action ，简化状态管理库
- 不需要嵌套模块，符合 Vue3 的 Composition api，让代码扁平化
- TypeScript支持
- 代码简洁，很好的代码自动分割

## 安装和使用pinia
npm i pinia

yarn add pinia

-> main.ts

-> store/index.ts
